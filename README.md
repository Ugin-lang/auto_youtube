# auto_youtube

Simple python script for fast opening YouTube 

How to install(all in linux term, any distro):

Create file "y" 
mkdir y
Insert script to file and save
Make file executable
chmod +x ~/y
Add to PATH:
mkdir -p ~/.local/bin
mv ~/y ~/.local/bin/y
Maybe reboot (optional)
DONE!

How to use:
1.open term
2.y [YOUR REQUEST]
3.YouTube just opened!
